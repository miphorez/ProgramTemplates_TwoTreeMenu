Программа предназначена для работы с контроллерами системы ИСБ "777" через COM-порт.
При работе через USB-COM адаптер необходимо, чтобы был установлен соответствующий драйвер.
Программа предназначена для работы в следующих операционных системах:
• Windows 7 (x86, x64)
• Windows 8 (x86, x64)
• Windows 10(x86, x64)
