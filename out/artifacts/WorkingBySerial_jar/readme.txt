Прежде чем запустить файл *.jar под вашей операционной системой, убедитесь, что у вас установлена Java. По ссылке https://www.java.com/ru/download/ вы всегда сможете скачать последнюю версию Java и проверить установку Java на вашем компьютере.
После установки java-программа запускается по двойному клику на файле *.jar или
через командную строку*: javaw -jar <имя файла>.jar

Под Windows проверку установки java можно произвести через командную строку. Для этого надо в терминальном окне* набрать: java -version
---------------------------
*В терминальное окно можно попасть через команду "cmd" в поле команд окна кнопки "Пуск" (в самом низу окна по кнопке "Пуск" - поле "Найти программы и файлы").
